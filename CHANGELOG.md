# Changelog

## [0.1.0] - 2026-03-11

### Added
- Live Feed with real-time file change tracking (create/modify/delete)
- Side-by-side Diff Viewer with syntax highlighting
- Stats Dashboard with animated counters, activity chart, and language donut chart
- Token estimation with Claude Opus 4, Sonnet 4, and Haiku 3.5 pricing
- File Heatmap showing most-modified files with heat-colored bars
- Session Replay engine with play/pause/stop and speed control (0.5x-5x)
- Animated File Tree showing workspace changes in real-time
- Ambient sounds via Web Audio API (optional, disabled by default)
- Export session as JSON, Markdown, or clipboard summary
- Status bar integration with live change count and lines/min
- Settings panel with toggles, sliders, and configuration options
- Theme-aware styling for both dark and light VS Code themes
- Activity Bar sidebar with mini dashboard and quick actions
- External browser window mode via HTTP + WebSocket server (for multi-monitor setups)

### Access Methods
- `Ctrl+Alt+B` — Open in external browser (recommended for multi-monitor)
- `Ctrl+Alt+M` — Open panel inside VS Code
- Activity Bar icon (eye) — Sidebar mini dashboard
- Editor title bar icons — Eye (internal) and link (external browser)
- Command Palette — All 7 commands available via `Ctrl+Shift+P`
- Status bar — Click to open panel
